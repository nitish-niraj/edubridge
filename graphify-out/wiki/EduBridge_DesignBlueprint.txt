Design Philosophy Overview
EduBridge has three completely distinct user groups � each with radically different needs, ages, and comfort levels with technology. The design system is split into three separate visual identities that share zero visual overlap, so each user feels the platform was built specifically for them.

?? STUDENT PORTAL
Ages 10�25
Bold, playful, Bloom-inspired. Warm corals, sunshine yellow, fresh mint. Bubbly rounded cards. Illustrated micro-details. Energetic and joyful.

?? TEACHER PORTAL
Ages 55�75
Extremely simple. Generous whitespace. Sage green + white. Very large text (18px base). No gradients. No animations. Every element labeled clearly.

?? ADMIN PORTAL
Platform team
Functional & efficient. Slate/blue-gray. Data-dense tables. Standard dashboard conventions. Dark sidebar, white content area.


SECTION 1 � STUDENT PORTAL  ??  Bloom Design System

Student Portal � Bloom Visual Identity
Inspired by the Bloom typeface from Google Fonts � a design language built for children and young learners. Warm, rounded, illustrated, energetic. Think colourful sticker-book meets modern edtech.

1.2  Typography
Design Element
Value / Token

Fredoka One (Google Fonts)

UI Font (headings)
Nunito (Google Fonts)
Import: Nunito:wght@400;600;700;800. Use for all h2�h4, nav labels, card titles. font-family: "Nunito", sans-serif
Body / Reading Font

Letter spacing
headings: -0.02em

1.3  Spacing, Shapes & Motion
Design Element
Value / Token
How to Implement in Blade + Vue

All cards, modals, panels. Use 20px. Feels bubbly and safe, not corporate.
Border radius � buttons

ALL buttons are pill-shaped. Never square buttons on student side.

transform: translateY(-4px) scale(1.01)
On hover, cards gently lift. Transition: all 0.25s cubic-bezier(0.34,1.56,0.64,1)
Button press animation
transform: scale(0.96)
On :active, buttons slightly shrink. Transition: 0.1s ease
Page transitions
Vue <Transition name="fade-up">
New pages slide up gently: translateY(20px) ? translateY(0), opacity 0?1, 0.3s
Illustrations
Undraw.co or custom SVG spots
Add small spot illustrations on empty states, success screens, onboarding. Use coral palette fill.
Grid layout
CSS Grid, 3 cols on desktop, 1 on mobile
Teacher cards in a 3-column grid. Gap: 24px. On tablet: 2 cols. On mobile: 1 col.
